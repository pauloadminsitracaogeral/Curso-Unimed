Realizado com um time de 25 pessoas, com carga horária de 16h, onde foram abordados vários temas:

- Importando Dados;
- Formatação de consulta;
- Ferramentas de dados; 
- Relacionamento;
- Linguagem M;
- Linguagem DAX;
- Medidas Calculadas;
- Criação de Dashboard;
- Publicando na web;
- Compartilhando.

E muito mais.
